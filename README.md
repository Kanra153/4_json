# Prettify JSON

Скрипт преобразует ваш неструктурированный JSON в удобный для чтения вид.

# Как использовать

Запуск на Linux, Python 3.5:

`$ python pprint_json.py path/to/file`

Пример работы скрипта:

```
"WorkingHours": [
            {
               "Hours": "09:30-22:30",
               "DayOfWeek": "понедельник"
            },
            {
               "Hours": "09:30-22:30",
               "DayOfWeek": "вторник"
            },
            {
               "Hours": "09:30-22:30",
               "DayOfWeek": "среда"
            },
            {
               "Hours": "09:30-22:30",
               "DayOfWeek": "четверг"
            },
            {
               "Hours": "09:30-22:30",
               "DayOfWeek": "пятница"
            },
            {
               "Hours": "09:30-22:30",

            },
            {
               "Hours": "09:30-22:30",
               "DayOfWeek": "воскресенье"
            }
         ],
 ```
